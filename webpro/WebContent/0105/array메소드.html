arr.push(a,b..) : 배열 끝에 요소추가 
arr.pop() : 마지막 요소 제거하고 리턴
arr.reverse() : 배열 뒤집기
arr.sort() : 사전순 정렬
arr.slice(start, end) : start ~ end-1까지 새로운 배열로 만들기





splice : 배열을 수정, 삭제
splice(3,2, 'A','B','C') : 3번째 위치에서 2개를 삭제하고 3번째 위치에 A,B,C를 삽입
splice(2,0, 'A','B','C') : 2번째 위치에서 삭제하지 않고 'A','B','C')
splice(4, 1) : 4번째 위치에서 1개를 삭제한다


concat : 여러개의 배열을 하나의 배열로 합니다.
join : 하나의 배열을 문자열로 변환한다.
