실행하자마자 가로와 세로의 길이를 입력한다
입력한 값으로 사각형의 가로 세로의 값으로 설정한다
확인 버튼 클릭하면 사각형의 정보를 출력한다

리터럴로 객체 생성
rect = {
			// 속성 속성 이름 : 속성값,
			width : garo,
			height: sero,
			// 함수선언
			getArea : function(){
				return this.width * this.height
			},
			getCircum: function(){
				return (this.width + this.height)*2
			}
			
};

rect객체 사용
rect.width
rect.height
rect.getArea()
rect.getCircum()

객체에 선언되지 않은 속성이나 함수를 동적으로 추가해서 사용할 수 있다.
