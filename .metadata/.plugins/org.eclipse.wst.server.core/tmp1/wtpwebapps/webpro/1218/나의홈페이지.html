html요소에는 block요소와 inline요소가 있다
		block요소는 한 줄 전체를 차지하고 자동으로 줄이 바뀐다
		- h1~6, p, ul, li, ol, table
		inline 요소는 지정한 크기만큼만 영역을 차지한다
		inline 요소는 가로 세로 크기를 지정할 수 없다(img 제외)
		a, img, span
